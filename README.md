 What is Git used for?
Git is a version control system used to track changes in code, manage project history, and collaborate with others without overwriting each other's work.
What is the difference between Git and GitHub?
Git is the tool that runs locally on your computer to track changes. GitHub is an online platform that hosts Git repositories so you can back them up, share them, and collaborate remotely.
 Why is branching useful?
Branching lets you work on new features or fixes separately from the main codebase, so you don't break working code while experimenting or developing something new.
What command did you use to upload your local commits to GitHub?
git push origin main
What did you learn from this activity?
